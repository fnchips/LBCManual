Config

1. Start the config program
『こん☆ふぃぐ』Launch [こん☆ふぃぐ] to reconfigure [LuckyBattleChronicle].
『こん☆ふぃぐ』After reconfinguring, please restart [LuckyBattleChronicle] to apply changes.
　
・Launch from shortcut


Double-click the shortcut created on desktop.

・Launch from Start Menu

Click the [Start] button and select [All programs] From the displayed list, click [LuckyBattleChronicle]. Click [こん☆ふぃぐ].

2. Configuration explanation




■ Key config
Only alphanumeric keys on the keyboard can be assigned
キー割り当てを行いたい項目を選択し、割り振りたいボタンを押すと自動的に反映されます。
選択終了後「オ～ケ～」ボタンを押すとゲームに反映されます。

■画面モード切替
ウィンドウモードとフルスクリーンモードの切り替えが可能です。
どちらかのモードを選択した後に「オ～ケ～」ボタンを押すと反映されます。

■音量調整
BGM、SEの音量調整が可能です。
BGM、SEそれぞれのバーで音量を調節してください。
「オ～ケ～」ボタンを押すとゲームに反映されます。
