Config

1. Start the config program
『こん☆ふぃぐ』Launch [こん☆ふぃぐ] to reconfigure [LuckyBattleChronicle].
『こん☆ふぃぐ』After reconfinguring, please restart [LuckyBattleChronicle] to apply changes.
　
・Launch from shortcut


Double-click the shortcut created on desktop.

・Launch from Start Menu

Click the [Start] button and select [All programs] From the displayed list, click [LuckyBattleChronicle]. Click [こん☆ふぃぐ].

2. Config window




■ Key setting
Only alphanumeric keys on the keyboard can be assigned
Select key to reassign control to the key you are pressing.
After reassigning keys, press [オ～ケ～] to save changes.

■ Screen setting
You can choose between windowed mode and fullscreen mode.
After choosing screen mode, press [オ～ケ～] to save changes.

■ Sounds setting
You can adjust the volume of BGM [Background music] and SE [Sound effect]
Use the sliders to increase/decrease the volume.
Press [オ～ケ～] to save changes.
