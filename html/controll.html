How to play

1．Controls
This game supports both keyboard and gamepad.
If you want to change the key bindings, make sure the game is not running and go to ［こん☆ふぃぐ］
※ Some keyboards may not support simultaneous input of three or more keys.


【 Default key settings 】
[ 1P ]
Jump：W
Left：A
Crouch：S
Right：D
Light attack：K
Heavy attack：L
Ready/Pause game：Space
　
[ 2P ]
Jump：UP
Crouch：DOWN
Left：LEFT
Right：RIGHT
Light attack：NumPad2
Heavy attack：NumPad3
Ready/Pause game：NumPad0

2．Moves
The moves are for 1P using the default keybinds and on the left side.


Move forward：D
Move back：A
Jump：W
Jump forward：D＋W
Jump backward：A＋W
Dash forward：Double tap D
Dash backward：Double tap A
Crouch：S
Light attack：K
Heavy attack：L
Special attack：D＋K
Throw/Grapple：K＋L (Press when you are next to opponent)
Block/Parry：Press A when opponent's attack reach you
Crouch block：Press A + S when opponent's attack reach you
There are attacks that can only be blocked by crouching blocked, and there are also attacks that cannot be blocked.
